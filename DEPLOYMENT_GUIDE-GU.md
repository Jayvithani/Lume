# લ્યુમ (Lume) પ્રોજેક્ટ ડેપ્લોયમેન્ટ ગાઇડ

તમારા પ્રોજેક્ટને લાઇવ કરવા માટે નીચેના સ્ટેપ્સ ફોલો કરો. મેં તમારા કોડમાં બે જરૂરી ફાઇલ્સ (`vercel.json` અને `netlify.toml`) અપડેટ/ઉમેરી છે, જેથી ડેપ્લોયમેન્ટમાં કોઈ એરર ન આવે.

---

## 1️⃣ બેકએન્ડ (Vercel પર ડેપ્લોયમેન્ટ)

તમારું બેકએન્ડ Node.js/Express માં બનેલું છે. Vercel પર ડેપ્લોય કરવા માટે આ સ્ટેપ્સ અનુસરો:

1. **GitHub પર કોડ પુશ કરો**: તમારા `bakend` ફોલ્ડરવાળા કોડને GitHub પર પુશ કરો.
2. **Vercel પર લોગિન કરો**: [vercel.com](https://vercel.com/) પર જઈને લોગિન કરો.
3. **નવો પ્રોજેક્ટ બનાવો**: "Add New..." બટન પર ક્લિક કરીને "Project" પસંદ કરો.
4. **GitHub રિપોઝિટરી કનેક્ટ કરો**: તમારી જે રિપોઝિટરીમાં કોડ છે તેને Import કરો.
5. **ફ્રેમવર્ક પ્રીસેટ (Framework Preset)**: એમાં "Other" અથવા "Node.js" પસંદ થયેલું જ હશે.
6. **રૂટ ડિરેક્ટરી (Root Directory)**: જો રિપોઝિટરીમાં આખું `lume` પ્રોજેક્ટ હોય, તો અહીંયા Edit પર ક્લિક કરીને `bakend` સિલેક્ટ કરો.
7. **Environment Variables**: જો ડેટાબેઝના કોઈ પાસવર્ડ અથવા `JWT_SECRET` હોય (જે `.env` ફાઇલમાં છે), તો તેને Vercel ની "Environment Variables" સેક્શનમાં ઉમેરો (દા.ત. `MONGO_URI`, `PORT`, `JWT_SECRET`).
8. **Deploy બટન દબાવો**: આનાથી તમારું બેકએન્ડ લાઇવ થઈ જશે અને તમને એક લિંક મળશે (દા.ત. `https://name.vercel.app`).

> **નોંધ**: મેં `vercel.json` ફાઈલને બરાબર સેટ કરી લીધી છે, જેથી તે તમારા `src/index.js` થી બરાબર રન થશે.

---

## 2️⃣ ફ્રન્ટએન્ડ (Netlify પર ડેપ્લોયમેન્ટ)

તમારું ફ્રન્ટએન્ડ Vite અને React થી બનેલું છે. આથી મેં તેમાં `netlify.toml` ફાઇલ એડ કરી છે, જેથી Page Refresh થાય ત્યારે એરર ન આવે.

1. **Vite ની લિંક બદલો**: તમારું બેકએન્ડ Vercel પર ડેપ્લોય થઈ જાય, એટલે જે નવી લિંક મળે તે લિંકને ફ્રન્ટએન્ડમાં જ્યાં પણ API call થાય છે (જેમ કે `axios` or `fetch` માં `http://localhost:5000` ની જગ્યાએ) ત્યાં બદલીને GitHub પર ફરી પુશ કરો.
   - *સારી રીત એ છે કે ત્યાં પણ `.env` ફાઇલ બનાવીને `VITE_API_URL` સેટ કરી શકો છો.*
2. **Netlify પર લોગિન કરો**: [netlify.com](https://netlify.com/) પર જઈને લોગિન કરો.
3. **નવી સાઇટ એડ કરો**: "Add new site" > "Import an existing project" પર ક્લિક કરો.
4. **GitHub કનેક્ટ કરો** અને તમારી Repository સિલેક્ટ કરો.
5. **બેઝ ડિરેક્ટરી અને કમાન્ડ**:
   - **Base directory**: `lume-fronted`
   - **Build command**: `npm run build`
   - **Publish directory**: `lume-fronted/dist` 
   *(ધ્યાન રાખજો કે ડિરેક્ટરીમાં જો તમે ખાલી `lume-fronted` સિલેક્ટ કરશો તો Netlify જાતે જ `dist` લઈ લેશે)*
6. **Deploy Site**: ડિપ્લોય પર ક્લિક કરો. થોડી વારમાં તમારી સાઇટ લાઇવ થઈ જશે!

---

## 🛑 અગત્યની વસ્તુ (CORS)
જ્યારે ફ્રન્ટએન્ડ Netlify પર લાઇવ થશે, ત્યારે તેને એક ડોમેન મળશે (જેમ કે `https://my-lume.netlify.app`). 
તમારા બેકએન્ડના `src/index.js` માં જ્યાં `cors()` સેટ કરેલું છે, ત્યાં આ નવી Netlify ની લિંક એડ કરવી જરૂરી છે, નહીંતર API કોલ બ્લોક થઈ જશે.

જો તમને આમાં કોઈ પ્રોબ્લેમ આવે અથવા કંઈ ન સમજાય તો મને પૂછી શકો છો!
